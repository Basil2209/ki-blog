Neuronales Netz

Ein neuronales Netz ist eine Ansammlung von miteinander verbundenen künstlichen Neuronen, die Informationen verarbeiten können. Ein künstliches Neuron ist eine vereinfachte Nachbildung eines biologischen Neurons, das ein oder mehrere Eingangssignale empfängt und ein Ausgangssignal erzeugt. Die Verbindungen zwischen den Neuronen haben Gewichte, die die Stärke der Übertragung bestimmen. Ein neuronales Netz lernt, indem es die Gewichte anpasst, um eine gewünschte Ausgabe zu erzielen.

Aufbau: Besteht aus mehreren Neuronen, sind durch Kanten verbunden, über welche sie Informationen übergeben.

1. Schicht: Eingabeschicht: Aufnahme von Informationen

2. Schicht: Verborgene Schicht: besteht aus beliebig vielen Ebenen. Informationen werden weitergegeben

3. Schicht: Ausgabeschicht: Geben die verarbeiteten Informationen aus

Funktion: Bilder Apfel und Birne unterscheiden. Farbpixel bestimmen und wie viele es in einem Bild hat und das wird als Information verwendet. Und diese Informationen werden gewichtet. Die Werte, die bei der verborgenen Schicht herauskommen, ob haben einen Wert und werden in eine Funktion eingesetzt. Mit einem Schwellenwert wird entschieden, ob ein Neutron reagiert und wie fest. Das dauert so lange, bis es auf der ausgebe Schicht einen Wert vorliegt. Um ein neuronales Netz zu trainieren, braucht es einen grossen Datensatz und viel Training. Das neuronale Netz wird ständig optimiert, bis es keine Fehler mehr macht.

Echtes Beispiel: Ein Beispiel für ein neuronales Netz ist ein Bilderkennungssystem, das ein Bild als Eingabe erhält und eine Kategorie als Ausgabe liefert. Das neuronale Netz besteht aus mehreren Schichten von Neuronen, die verschiedene Merkmale des Bildes extrahieren und kombinieren. Die erste Schicht kann zum Beispiel einfache Kanten erkennen, die nächste Schicht kann Formen erkennen, und so weiter. Die letzte Schicht kann dann eine Entscheidung treffen, ob das Bild eine Katze, einen Hund oder etwas anderes zeigt. Diese Beispiele zeigen, wie neuronale Netze komplexe Probleme lösen können, indem sie aus Daten lernen und Muster erkennen.
